Sobre Angular

Angular es un framework de desarrollo web, permite crear interfaces de usuario dinámica, facilitando la interacción del usuario con la aplicación sin necesidad de recargar la página. Angular dispone de herramientas para gestionar el estado de la aplicación, la manipulación del DOM y la comunicación con servicios backend.

Angular se utiliza en una amplia variedad de aplicaciones, tales como:

  • Aplicaciones web empresariales
  • Aplicaciones web empresariales
  • Aplicaciones móviles híbridas
  • Aplicaciones de comercio electrónico.
  • Aplicaciones de gestión de contenido
  • Proyectos que requieren una experiencia de usuario interactiva y fluida

Angular se basa principalmente en TypeScript, que es un superconjunto de JavaScript. TypeScript añade tipado estático y otras características que facilitan el desarrollo a gran escala. Además, Angular utiliza HTML para la estructura de las vistas y CSS para el estilo de las aplicaciones.

Para poder hacer uso del framework es necesario disponer de:

  1. Node.js v18.19.1 o superior
  2. Editor de texto (Se recomienda Visual Studio Code)
  3. Un terminal (necesario para correr los comandos CLI de Angular)

Es necesario instalar Angular en nuestro PC, pero antes de nada vamos a asegurarnos que la versión que hemos instalado es adecuada, para ello ejecutamos el siguiente comando desde la terminal:

Primeros Pasos

Con nuestro proyecto levantado podemos empezar con el código de nuestra web, para ello vamos a ayudarnos del editor de texto Visual Studio Code:

code --ruta del proyecto--

Una vez abierto podemos ver que se despliega un gran número de ficheros, antes de ponernos manos a la obra vamos echar un ojo.

La captura que se muestra continuación son los ficheros que nos permitirán trabajar con la web:

  • index.html: es el archivo donde escribiremos nuestro código HTML y donde incluiremos los "tags" de los diferentes componentes.
  • main.ts: fichero donde importaremos los componentes
  • styles.css: fichero donde vamos a definir los estilos de nuestra web

Otro conjunto de fichero son lo que conforman "app". App es un componente, pero... ¡¿QUE ES UN COMPONENTE?!

En Angular, un componente es una combinación de una plantilla HTML, estilos CSS y lógica TypeScript. Representa una parte de la interfaz de usuario y puede tener su propio comportamiento y datos, pudiendo añadir esa plantilla en cualquier momento y lugar dentro de nuestro index.html.
